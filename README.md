# expo-lottie

Sample Expo (react-native) application which uses Lottie library

> ⚠️ `anim_V3.json` crash on android.
> https://github.com/expo/expo/issues/8011

## To test

```sh
git clone https://github.com/maxgfr/expo-lottie.git
cd expo-lottie
expo install
expo start
```
